Process Page

Planning Stage

Starting this Project, I looked at the Project 1 requirements, and I recognize that I need to pick a Wikipedia page with sufficient content. Then I need to analyze the page and map out the information architecture of the page. Then I need to find a public Canadian university website to base the design of the pages. Then I need to create 3 different pages, a home page, a sub-page, and a process page using HTML and CSS.


I chose to use the Web Design Wikipedia page because it had a decent amount of content. Also, I chose to use the University of Waterloo's website as a reference for the design of the pages. I also decided to create a Web Design History sub-page based on the University of Waterloo's Admission's sub-page.

Information Architecture

I analyzed and mapped out the Structural, Associative, and Utility navigations for the original web design Wikipedia page. I used gloomaps online information architecture mapping tool to map the Wikipedia page. The yellow blocks represent the 1st level item of each type of navigation. Then the orange blocks represent the sub-level of the yellow blocks, making it the 2nd level element. Lastly, the blue blocks represent the sub-level of the orange blocks, making it the 3rd level item. I chose to use the web design Wikipedia because it had a decent amount of content as seen in the Information Architecture image below.

I created a new information architecture for the Web Design Wikipedia page. I used the same mapping tool, gloomaps, to map out the new information architecture. I chose to use the University of Waterloo's website as a reference for the design of the pages. Most of the navigation is based on the University of Waterloo's website. Then I just replaced the navigational content from the University of Waterloo website with the content from the Web Design Wikipedia page. Additionally, the green blocks represent the sub-level of the blue blocks, making it the 4th level item.

Wireframing

The image below is the homepage of the University of Waterloo's website that I used as reference to create the wireframe. The current University of Waterloo's website is different because they recently updated it when I was creating the website. The website uses a spotlight image to display latest content of the page. They used a lot of white space and a lot of images to make the website look more modern. They also used a lot of content cards to display the content like news, facts, and events.

I created a wireframe for the home page, the sub-page, and the process page using Figma. For the homepage, I added the spotlight image and the content cards to display the content. I also have a navigation bar at the top of the page and a footer at the bottom of the page. For the History sub-page, I added a spotlight image and more images to accopany the content. As well as headers with bottom borders to seperate the content. To represent the different hierarchy of the content, I used different font sizes and different background colors for the different sections.
